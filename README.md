#  SneakHype — Ecommerce Project using Flask

## Overview
SneakHype is an intermediate-level e-commerce platform dedicated to footwear, built using **Python** and **Flask**. It delivers a holistic shopping journey featuring **user authentication**, **product management**, **shopping cart**, and **order processing**—all wrapped in a clean, maintainable codebase.  
[GitHub Repository](https://github.com/Ayush-Shah96/SneakHype---Ecommerce-Project-using-Flask)

---

## Features That Stand the Test of Time
- **Footwear Catalog & Product Management** – Showcase and manage products clearly and effectively.  
- **User Authentication** – Secure registration, login, logout—pillars of trust.  
- **Shopping Cart** – Add to cart, update, and remove items with ease.  
- **Order Processing** – Streamlined checkout flow and order tracking.  

---

## Tech Stack — Built on Core Principles
- **Backend:** Python with Flask  
- **Frontend Templates:** HTML (via Flask’s template engine)  
- **Database:** SQLite (`footwear_store.db`)  
- **License:** MIT

## Getting Started 

Clone the Repository
git clone https://github.com/Ayush-Shah96/SneakHype---Ecommerce-Project-using-Flask.git


Set Up a Virtual Environment (recommended best practice)
python3 -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate


Install Dependencies
pip install flask
pip install -r requirements.txt


Run the Application
python app.py


Then navigate to http://localhost:5000 to bring your classic yet forward-looking store to life.
